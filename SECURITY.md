# Security Policy

## Supported versions

Ascend is pre-release software. No public version is currently supported for production use.

## Reporting a vulnerability

Use GitHub's private vulnerability reporting for this repository:

https://github.com/cosmicbubble898/ascend/security/advisories/new

Do not disclose a suspected vulnerability, exploit details, credentials, recordings, or user data in a public issue.
Include the affected version or commit, reproduction steps, impact, and any suggested mitigation in the private report.

This project does not currently offer a bug bounty or guaranteed response time.
